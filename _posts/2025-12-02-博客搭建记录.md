## 摘要
- 本网站基于 Jekyll 静态网站生成器，托管于 GitHub Pages，主题为默认。
- 主要搭建流程：
  1. 初始化 GitHub 仓库并配置 GitHub Pages。
  2. 本地安装 Ruby 和 Jekyll 环境，创建新站点。
  3. 配置 `_config.yml` 文件，设定站点信息及插件需求。
  4. 应用并自定义 Minima 主题样式，调整导航、标签、分类等功能。
  5. 使用 Markdown 编写文章，按需添加自定义页面（如 About、Tag）。
  6. 推送更新至 GitHub，自动触发构建和部署至个人域名。
- 整体流程简单高效，适合个人技术博客的长期维护与扩展。
